Polygon Maker

Creates polygons on top of images

SOFTWARE

Polygon Maker

By Hugo Belin

December 15, 2020

This post allows you to draw polygons on top of a picture of your choice. This comes handy for several scenarios, in our case we have used it to get path coordinates for Tableau visualizations. With the path coordinates we were able to draw the resulting polygons on top of the images in Tableau with the ability to correlate this data for zones or areas that we wanted. To view the usage sample check out this post.

Usage

To make use of this utility follow these steps:

  • Copy the path of the image on the web (we’re using this sample from Unsplash )

Volcanoes

  • Click the Load button to put it on the canvas
  • The coordinate system start from left to right and top to bottom (upper left corner is 0,0), if you want it to be different you can use the offset input fields provided
  • Every vertex (where you click on the picture) is denoted by a square (default size is 10 pixels), the size field allows you to change the size of the squares
  • Checkbox field works with tags, if you use right click you can use tags, if you add a tag and the checkbox is active further left clicks will use the last tag
  • Once you loaded the image on the canvas Start button will start recording clicks on the image and the coordinates will be appended to the bottom of the image. Right clicks can be used to add tags to the points which can be used to denote regions

Path Outline

  • Once the desired coordinates are recorded at the bottom use the Download link to get them via CSV
  • If you desire to start over use the Clear button
