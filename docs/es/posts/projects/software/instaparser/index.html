Instaparser

Utilidad básica para encontrar y extraer IDs de fotos de Instagram, estos IDs provienen de la URL de la foto con el fin de embeber esos artículos

Instaparser

By Hugo Belin

August 8, 2020

Note: Este contenido es obsoleto y lo hemos dejado como nota de referencia

Sencilla utilidad para obtener ids the fotos de Instagram. Permite embeber fotos de Instagram en nuestra página web la cuál puede ser a su vez embebida en otros lugares (como Vizzes de Tableau 😉 )

Instrucciones:

  • Copie el URL de una foto de Instagram e ingréselo en el campo de abajo
  • Podrá ver el fragmento que identifica la foto que puede ser usado con nuestro sitio especial de Instagram , puede agregar el siguiente fragmento a la URL ?insta=<postId> como se muestra
  • Obtendrá también un enlace al sitio de Instagram con el fragmento añadido el cuál puede abrir. Siéntase en libertad de embeber ese sitio en otros lados
